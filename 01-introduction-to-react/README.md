# 📌 What is React?

React is a popular **JavaScript library** used to build modern and interactive **User Interfaces (UI)** for websites and web applications.

It was created by Facebook in 2013 and is now maintained by Meta.

React is one of the most demanded frontend technologies in today’s job market.

# ❓ Why React Was Created?

Before React, developers mainly used:

* HTML
* CSS
* JavaScript

But large applications had many problems:

### ❌ Problems in Traditional Development

* Full page reload
* Slow performance
* Too much DOM manipulation
* Repeated code
* Hard to maintain UI
* Poor user experience

React was introduced to solve these issues.

---

# ⚡ How React Solves These Problems

## ✅ Virtual DOM

React uses **Virtual DOM**.

Instead of refreshing the whole page, React updates only the changed part.

### Example:

If notification count changes:

* Old Method → Full page reload
* React → Only notification area updates

This improves speed and performance.

---

## ✅ Reusable Components

React allows developers to break UI into small reusable parts called components.

### Examples:

* Navbar
* Footer
* Sidebar
* Button
* Product Card
* Login Form

Create once, use many times.

---

## ✅ Single Page Application (SPA)

React applications mostly work as SPA.

### Meaning:

* Page does not fully reload
* Only content changes
* Faster navigation
* Better user experience

# 🧠 Important Terms in React

## JSX

JSX stands for JavaScript XML.

It looks like HTML but works inside JavaScript.

```jsx
const element = <h1>Hello React</h1>;
```

---

## Component

A reusable function that returns UI.

```jsx
function Header() {
  return <h1>Welcome</h1>;
}
```

---

## Props

Used to pass data from parent component to child component.

---

## State

Used to store changing data like:

* Counter value
* Form input
* Toggle button
